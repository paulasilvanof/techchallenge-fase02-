# techchallenge-fase02-
Repositório do Tech Challenge com os entregáveis da Fase 02.

O objetivo é desenvolver modelos de Machine Learning capazes de prever se um vinho possui alta qualidade ou qualidade média/baixa com base em características físico-químicas.

Fonte:
Wine Quality Dataset (UCI/Kaggle) https://www.kaggle.com/datasets/arnavs19/wine-quality-uci-machine-learning-repository

Variável alvo:
quality_binary

0 = Baixa/Média Qualidade
1 = Alta Qualidade

Modelos utilizados:
- K-Nearest Neighbors (KNN)
- XGBoost

Principais resultados:
- Álcool apresentou maior correlação positiva com a qualidade.
- Acidez volátil apresentou maior correlação negativa.
- XGBoost apresentou melhor desempenho.

Como executar:
pip install -r requirements.txt
